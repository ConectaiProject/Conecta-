Sobre Nós 🐼

O Conectaí é um projeto desenvolvido em parceria com a Serasa Experian e o Instituto PROA, com o objetivo de facilitar a inserção de pessoas com mais de 40 anos no mercado de trabalho.

A plataforma foi criada para tornar esse processo mais acessível, oferecendo tutoriais simples, orientação prática e acesso a vagas exclusivas, tudo pensado para atender às necessidades desse público.

Desenvolvemos o Conectaí a partir do entendimento de que muitas pessoas enfrentam dificuldades ao buscar novas oportunidades profissionais após certa idade. Por isso, buscamos criar uma solução que promova inclusão, autonomia e novas possibilidades de crescimento.

FERRAMENTAS UTILIZADAS 🦝

**Visual Studio Code (VSCode):** utilizado para a programação do projeto, com as linguagens HTML, CSS e JavaScript, responsáveis pela estrutura, estilo e interatividade do site.

**Figma:** empregado na prototipação e no desenvolvimento do layout, garantindo uma interface intuitiva e acessível.

**Canva:** utilizado na criação de elementos visuais e apoio ao design, contribuindo para a identidade visual do projeto.
